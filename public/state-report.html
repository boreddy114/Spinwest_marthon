Technical Architecture & Infrastructure Report

Prepared For: State of Colorado / CO4Kids Organization

1. Executive Summary

This document outlines the technical infrastructure, software architecture, and cloud billing requirements for the custom CO4Kids Photo Kiosk application developed specifically for the Resource Fair. The application is designed to securely capture high-quality event photos via iPad/Mobile devices, upload them efficiently to a cloud database, and deliver them directly to attendees' inboxes with high deliverability rates.

2. Technical Architecture

The application was built using modern, reliable, and scalable web technologies to guarantee a seamless experience for both attendees and event volunteers.

Frontend Framework: Next.js (React)

  • Provides a fast, responsive, and application-like experience directly in the web browser.
  • Integrates directly with the device's native camera hardware through modern WebRTC APIs, supporting dynamic camera flipping and live digital zooming.
  • Includes a dedicated "Volunteer Mode" to rapidly capture and securely store event memories without bottlenecking the attendee line.

Backend Storage: Supabase

  • Acts as the primary secure database and object storage pipeline.
  • Instantly processes base64 image strings from the camera hardware, converts them into optimized JPEG files, and generates secure public URLs for the event gallery mapping.

Mail Delivery Pipeline: NodeMailer & State SMTP

  • To ensure photos do not get flagged as spam or phishing attempts, the application's outbound email pipeline was explicitly programmed to authenticate directly with official @state.co.us credentials.
  • The system dynamically constructs HTML emails containing the attendees' photos, effectively bypassing automated spam filters that traditionally block automated photo attachments.

3. Cloud Infrastructure & Billing Justification

To safely handle the high volume of traffic, high-resolution photo uploads, and concurrent user sessions expected during the live Resource Fair, the underlying cloud infrastructure required upgrading to Enterprise/Pro tiers. Free tiers on cloud providers impose strict rate limits and storage caps that would disrupt the event.

Service Plan Cost Purpose & Necessity
Vercel Pro Plan $20.00 Hosts the foundational application and Edge Functions. The Pro plan guarantees elastic auto-scaling under heavy event traffic, ensuring zero throttling, zero offline queues, and enhanced security.
Supabase Pro Plan $25.00 Handles all image storage and database logging. Upgrading to Pro expands the persistent file storage ceiling to 100 GB (accommodating tens of thousands of high-quality event photos) preventing the kiosk from pausing or rejecting uploads due to storage exhaustion mid-event.
Total Required Infrastructure Spend: $45.00 / month
